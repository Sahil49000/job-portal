# 💼 Job Portal System

A full-stack Job Portal application that allows users to search and apply for jobs, while administrators can manage job listings efficiently.

---

## 🚀 Features

### 👤 User (Job Seeker)

* Register and login securely
* Search jobs by role, location, or keywords
* View job details
* Apply for jobs
* Track applied jobs

### 🛠️ Admin

* Secure admin login
* Add new job postings
* Update job details
* Delete job listings
* View applicants

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/job-portal.git
cd job-portal
```

### 2️⃣ Setup Backend

```bash
# Example (Node.js)
npm install
npm start
```

### 3️⃣ Setup Database

* Import `schema.sql` into MySQL/SQLite
* Configure DB connection in backend

### 4️⃣ Run Frontend

* Open `index.html`
  OR
* Run using Live Server in VS Code
